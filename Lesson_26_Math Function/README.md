# Lesson 26 - Math Functions

In this lesson, we learn about the **math functions** in C++ that are available in the `<cmath>` library.  
They allow us to perform mathematical calculations easily.

## Key Points
- **`sqrt(x)`** â†’ returns the square root.  
  Example: `sqrt(25)` â†’ 5
- **`pow(x, y)`** â†’ raises x to the power of y.  
  Example: `pow(2, 3)` â†’ 8
- **`abs(x)`** â†’ returns the absolute value.  
  Example: `abs(-10)` â†’ 10
- **`round(x)`** â†’ rounds to the nearest integer.  
  Example: `round(4.6)` â†’ 5
- **`floor(x)`** â†’ rounds down.  
  Example: `floor(4.9)` â†’ 4
- **`ceil(x)`** â†’ rounds up.  
  Example: `ceil(4.1)` â†’ 5

Math functions are often used in **algorithms, games, and real-world applications**.
