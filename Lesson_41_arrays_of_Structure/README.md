# Lesson 41 - Arrays of Structures  

In this lesson, we learn about **arrays of structures** in C++.  
Arrays of structures allow us to store multiple instances of a structure in a single array, making it easier to manage related data.

## Key Points
- **Definition** â†’ An array where each element is a structure.  
  Example: `struct Student students[5];` stores 5 `Student` structures.
- **Access Members** â†’ Use the index to select an element, then the dot operator to access its members.  
  Example: `students[0].name = "Ali";`
- **Initialization** â†’ Each structure in the array can be initialized separately.  
- **Use Cases** â†’ Storing records, lists of objects, or collections of related data.
- **Benefits** â†’ Organizes complex data and allows batch operations on multiple objects.
